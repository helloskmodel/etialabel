# ETIA 内容填空模板 · Content Fill-in Templates

> 用法:每个行业/产品/笔记**复制一份**下面对应模板,把 `[...]` 填掉即可。
> 先填英文(源语言);中/泰/越后补,缺的自动回退英文。**没有的参数写 `待核实`,别编。**
> 填好发我,我转成 `data/` —— 你不用碰代码。

---

## 模板 A · 行业落地页(By Industry / Environment / Material 通用)

```
行业 slug(英文):        [pcb]
Hero 标题:              [PCB Labelling Solutions]
Hero Slogan(一句):      [.....]
Banner 图(COS slug):     [industries/pcb]   → /img/industries/pcb.jpg

── Part 1 · 行业介绍(300–400字)──────────────
[这个行业的标识挑战 + ETIA 方案概述,300–400 字]

── 产品列表(FlexCon Bar)──────────────
是否两层分类? [否 / 是]
若「是」,一级按钮:      [一级A] [一级B] [一级C]

Tab(每个产品一条,想列几条列几条):
  Tab1  产品名:[.....]  图slug:[.....]  说明(15–30字):[.....]
  Tab2  产品名:[.....]  图slug:[.....]  说明:[.....]
  Tab3  ...

── 用途卡 ×1 ──────────────
标题:[Label 主要用途]   说明(40–80字):[.....]

── 产品卡(1–3 张并列)──────────────
  卡1  产品名:[.....]  一句话:[.....]  链接到产品页:[/products/item/xxx/]
  卡2  ...
  卡3  ...
```
