## AN-E1 · SMT 回流焊 / 波峰焊 高温标签

**核心关键词：** 回流焊标签；波峰焊标签；耐高温 PCB 标签；聚酰亚胺标签
**建议 URL：** `/zh/labels/smt-reflow-wave-solder`

### 应用概述
PCB 在组装、测试到后期维修的全过程中，标签必须承载追溯身份。无铅回流焊、波峰焊、强活性助焊剂与高压化学清洗轮番作用，标签要在这一切之后仍保持条码清晰可扫。

### 应用挑战
普通标签在回流焊后**收缩、翘起或变黄**，条码无法识别，追溯就此中断——而这恰恰发生在最关键的制程节点。

### 性能要求
- 耐峰值 **300°C（572°F）** 多次过炉
- 耐**免洗 / 水溶性 / 松香型助焊剂**与波峰锡蚀
- 耐高压水洗 / 化学清洗（IPA、MEK、水基清洗剂）
- 面向细字与高密度 **Data Matrix / 二维码**
- 对高表面能 PCB 基材永久粘接

### ETIA 方案（Polyonics® Apex 系列）
采用 Polyonics® **Apex 系列**聚酰亚胺（哑面 / 半光 / 亮面可选），配**新一代专有涂层**——不泛黄、不软化、耐磨，助焊剂耐受较上一代**+72%**、清洗剂耐受 **+15%**。亮面 XF-101G 适合高密度二维码，超薄 XF-616 适合 BGA / 细间距。

|---|---|---|---|---|
| **XF-101M / XF-102M** | 哑白 | 1 / 2 mil | 最高 300°C | 通用低厚度 PCB 标识 |
| **XF-101S / XF-102S** | 半光白 | 1 / 2 mil | 最高 300°C | 扫码与操作平衡 |

### 客户收益
- **条码挺过整条产线** —— 300°C 多次过炉后仍可靠粘接、可扫，追溯不断链
- **清洗更狠也不糊印** —— +72% 助焊剂 / +15% 清洗剂耐受，图像保真
- **贴得进密板** —— 超薄低厚度避开低矮元件与密集布局
- **选对表面** —— 固定扫码选亮面，防眩选哑面，操作平衡选半光

## AN-E2 · ESD 防静电 PCB / 元器件标签

**核心关键词：** ESD 标签；防静电标签；静电耗散标签；ANSI/ESD S20.20 标签
**建议 URL：** `/zh/labels/esd-static-dissipative`

### 应用概述
在 ESD 防护区（EPA）内，凡接触或靠近敏感电子的物品都必须静电安全——标签也不例外。它既要管理电荷，又要是耐用、可打印的耐高温标识。

### 应用挑战
普通标签会**积累又释放静电**，损坏敏感半导体与 IC，反而成为 EPA 要消除的隐患。

### 性能要求
- 表面电阻处于**耗散范围（10⁶–10¹¹ Ω；XF-781 族 10⁹–10¹¹ Ω）**
- 符合 **ANSI/ESD S20.20**（及 ESD S541 / IEC 61340 / JEDEC JESD 625B 体系）
- **低起电**（离型剥离 / 重贴 < 125 V）
- 耐 **260–300°C 回流焊**、无卤
- 静电耗散**胶层与离型层**（全链路电荷管理）

### ETIA 方案（Polyonics® Apex ESD）
采用 Polyonics® **Apex ESD** 聚酰亚胺——静电耗散胶 + 静电耗散离型层，**低起电 PSA（<125 V）**、无卤，符合 ANSI/ESD S20.20。半光 XF-101SE / 哑面 XF-101ME 满足不同扫码与防眩需求；XF-781/782 为文档最完整的 ESD 半光款。

### 推荐产品
| 型号 | 表面 / 类型 | 厚度 | 耐温 | 说明 |
|---|---|---|---|---|
| **XF-101SE / XF-102SE** | ESD 半光白 | 1 / 2 mil | 最高 300°C | Apex ESD 主力 |
| **XF-101ME** | ESD 哑白 | 3 mil | 最高 300°C | 防眩、较厚结构 |
| **XF-781 / XF-782** | ESD 半光白 | 1 / 2 mil | −40~300°C | 10⁹–10¹¹ Ω，UL969、无卤 |

### 客户收益
- **保护敏感器件** —— 标签管理电荷，不在 EPA 内成为放电源
- **通过 ESD 审核** —— 耗散面材 + 胶层，符合 ANSI/ESD S20.20 体系
- **过得了回流** —— 耐 300°C，ESD 与耐高温兼得
- **低起电、无卤** —— 剥离 / 重贴 <125 V，满足敏感器件洁净要求

## AN-E3 · UL94 V-0 阻燃标签

**核心关键词：** 阻燃标签；UL94 V-0 标签；VTM-0 阻燃；电子阻燃标识
**建议 URL：** `/zh/labels/ul94-v0-flame-retardant`

### 应用概述
电子设备、电池、线束与电气元件常要求阻燃标识，以满足产品安规与消防要求。标签在起火时不能助燃或助长火势蔓延。

### 应用挑战
非阻燃标签会**助长火势蔓延**、产生烟气，导致产品无法通过 UL / 安规认证。

### 性能要求
- **UL94 V-0 / VTM-0** 阻燃等级
- 耐温、电气绝缘、低烟
- 兼容电子制程（回流 / 清洗）
- 可打印、可追溯

### ETIA 方案（Polyonics® 阻燃聚酰亚胺）
采用 Polyonics® **XF-603 阻燃聚酰亚胺**（符合 UL94 VTM-0，耐 300°C，对标 Brady B-8088A）；较低温 / PET 需求可选 **XF-611 阻燃 PET**（UL94 VTM-0，对标 Brady B-5109FR）。

### 推荐产品
| 型号 | 材料 | 厚度 | 耐温 | 阻燃 | 说明 |
|---|---|---|---|---|---|
| **XF-603** | 聚酰亚胺 PI | 1 mil | 最高 300°C | **UL94 VTM-0** | 电子阻燃 PCB 标识 |
| **XF-611** | 聚酯 PET | 1.5 mil | 最高 150°C | **UL94 VTM-0** | 较低温 / 线缆阻燃 |

### 客户收益
- **过安规** —— UL94 V-0/VTM-0 阻燃，助力产品认证
- **不助燃、低烟** —— 降低火势蔓延与烟气风险
- **兼顾耐温** —— PI 款耐 300°C，回流制程可用
- **可追溯** —— 阻燃同时保持条码 / 标识清晰
