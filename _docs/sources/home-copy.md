# DESIGN DIRECTION

## Hero
- Large editorial-style typography; white / very light background; one strong industrial application image.
- Body copy within two short lines on desktop. Primary CTA: Explore Applications. Secondary: Request Samples.

## Service Bar
- Four short items only, no descriptions, simple line icons. May be hidden on mobile if it interrupts flow.

## Why ETIA
- Four-column desktop layout: We Understand / We Match / We Engineer / We Deliver.
- Each: one icon + the "We…" heading + no more than two short lines. Mobile: horizontal swipe or two-column grid.

## Application Section
- Large image-and-copy CAROUSEL, not six small cards.
- Each slide: large industry image · industry name · application headline · one short description · three performance tags · one featured material/solution · View Application →
- Desktop: image ~55% / copy ~45%; show carousel numbering e.g. 01 / 06.
- Mobile order: image → industry → headline → three tags → featured material → CTA.
- Products appear only as supporting proof within each slide. The application remains the primary navigation path.
