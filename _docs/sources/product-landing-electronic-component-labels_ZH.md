# 为极端环境而生的电子元器件标签

Polyonics® 是一家特种**涂层制造商 —— 而非标签印刷厂。** 我们研发的是让电子元器件标签能挺过 **300°C（572°F）回流焊**、波峰焊、强活性助焊剂以及整个 PCB 组装工艺的*材料*。**Apex 系列**的助焊剂/焊接耐受性比上一代**提升 72%**。

**ETIA** 供应这些 Polyonics 材料 —— 以**转换卷材**（供自行印刷）或**完整规格 OEM 材料**形式提供 —— 让正在评估材料方案的 OEM、代工厂与转换商在这里获得完整信息，包括 Polyonics 与目录型供应商到底有何不同。

## 为 PCB 组装工艺而生

电子元器件与 PCB 标签在电路板投入使用之前，就要在其一生中最严苛的工序里承载追溯身份。一张标签可能面对：

- 最高 300°C 的**多次回流焊**与**波峰焊**
- **强活性助焊剂** —— 免洗、水溶性与松香型
- **高压化学清洗** —— IPA、MEK 与水基清洗剂
- 搬运、组装与现场暴露中的磨损

面材、涂层、胶粘剂与离型层作为一个整体，决定了条码在最后是否仍可读。这是一个**涂层问题** —— 也正是 Polyonics 所专注的工程。
