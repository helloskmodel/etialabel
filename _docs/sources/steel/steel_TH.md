## 2. ทำไมอุตสาหกรรมเหล็กจึงต้องใช้ฉลากทนความร้อน

สิ่งที่ทำให้การผลิตเหล็กแตกต่างคือ **ณ วินาทีที่ต้องระบุตัวตนวัสดุ ชิ้นงานมักยังร้อนแดง มีสเกลออกไซด์ปกคลุม และกำลังจะเข้าสู่กระบวนการที่รุนแรงกว่าเดิม**

ฉลากทั่วไปล้มเหลวทั้งหมดในสภาวะนี้:

- **ฉลากกระดาษ / PET** — ไหม้เป็นถ่านเมื่อเกิน 200℃
- **การพ่นรหัส / สี** — เกาะอยู่บนชั้นสเกล เมื่อสเกลหลุดร่อนเครื่องหมายก็หายไปด้วย และถูกเผาหมดหลังผ่านเตาอบ
- **การแขวนป้ายด้วยมือ** — พนักงานต้องเข้าใกล้แหล่งความร้อน 700℃ ช้าและไม่ปลอดภัย
- **การตอกตัวเลข / ตอกจุด** — เครื่องอ่านไม่ได้ เชื่อมต่อ MES แบบเรียลไทม์ไม่ได้

คุณค่าของฉลากทนความร้อนคือการทำให้ **บาร์โค้ดยังคงอ่านด้วยเครื่องได้ตลอดทั้งห่วงโซ่การแปรรูปร้อน** เชื่อมโยงวัสดุจริงเข้ากับระบบสารสนเทศอย่างแท้จริง

## 3. ประเภทการใช้งานฉลากในอุตสาหกรรมเหล็ก

### ประเภทที่ 1: ฉลากติดโดยตรงขณะร้อน (Direct Hot Application)

- **ใช้ที่ไหน:** สแลบและบิลเล็ตที่เพิ่งออกจากเครื่องหล่อต่อเนื่อง คอยล์หลังการรีดร้อน
- **ความท้าทายหลัก:** อุณหภูมิผิวขณะติดอยู่ที่ 550–750℃ ผิวหยาบและมีสเกลปกคลุม ฉลากต้องยึดติดทันทีและต้องไม่ไหม้
- **ทำงานอย่างไร:** ระบบหุ่นยนต์ติดฉลากทำงานอัตโนมัติข้างโต๊ะลูกกลิ้ง — พนักงานไม่ต้องสัมผัสแหล่งความร้อนเลย
- **แก้ปัญหาอะไร:** วัสดุร้อนมีตัวตนเฉพาะตั้งแต่วินาทีที่ผลิตออกมา ไม่ต้อง "จำตำแหน่ง" หรือ "เดาหมายเลขเตา" อีกต่อไป

### ประเภทที่ 2: ฉลากติดตามตลอดกระบวนการอบชุบ (Through-Process Labels)

- **ใช้ที่ไหน:** การกัดกรดและอบอ่อนลวดเหล็ก / เหล็กเส้น การตีขึ้นรูปร้อนและการอบอ่อนระยะยาวของชิ้นงานตีขึ้นรูป
- **ความท้าทายหลัก:** ฉลากไม่ได้แค่ "ทนร้อนชั่วครู่" แต่ต้อง **แช่กรดก่อน (H₂SO₄ / HCl) แล้วเข้าเตาอบนาน 10–18 ชั่วโมง** การกัดกร่อนทางเคมีและความร้อนระยะยาวซ้อนทับกัน
- **ทำงานอย่างไร:** ติดที่อุณหภูมิห้อง ฉลากเดินทางไปกับชิ้นงานตลอดห่วงโซ่การอบชุบ และยังสแกนได้เมื่อออกจากเตา
- **แก้ปัญหาอะไร:** การอบชุบคือขั้นตอนที่สูญเสียการระบุล็อตบ่อยที่สุด ถ้าฉลากรอด = การสอบกลับไม่ขาดตอน

### ประเภทที่ 3: ป้ายแขวนทนความร้อน (Tags)

- **ใช้ที่ไหน:** จิ๊ก ฟิกซ์เจอร์ ตะกร้า และแม่พิมพ์ที่ใช้หมุนเวียนซ้ำ
- **ความท้าทายหลัก:** ไม่ใช่การใช้ครั้งเดียว — ต้องอ่านได้หลังผ่าน **รอบความร้อนหลายสิบถึงหลายร้อยรอบ**
- **ทำงานอย่างไร:** ยึดเชิงกล (มัดลวด / ย้ำหมุด) เข้ากับสินทรัพย์เพื่อติดตามระยะยาว
- **แก้ปัญหาอะไร:** การจัดการสินทรัพย์จิ๊ก การนับจำนวนรอบการใช้งาน การแจ้งเตือนหมดอายุการใช้งาน

### ประเภทที่ 4: การพิมพ์หน้างานและการเชื่อมต่อระบบ

- **ใช้ที่ไหน:** ครอบคลุมทุกกรณีข้างต้น
- **เนื้อหา:** เครื่องพิมพ์ถ่ายโอนความร้อน + ริบบอนทนความร้อน MES ส่งหมายเลขเตา / หมายเลขล็อตแบบเรียลไทม์ พิมพ์แล้วติดทันที สถานีสแกนปลายทางส่งข้อมูลกลับ

## 2. การใช้งานเฉพาะ (Application)

**สถานการณ์:** บิลเล็ตที่ถูกตัดจากเครื่องหล่อต่อเนื่องมีอุณหภูมิหน้าตัด / ผิวข้างประมาณ 750℃ ผิวปกคลุมด้วยสเกลออกไซด์ที่หลุดร่อนอยู่ตลอด บิลเล็ตต้องรอบนเตียงระบายความร้อน ถูกขนย้าย จัดกอง และสุดท้ายเข้าเตาอบหรือส่งตรงไปยังเครื่องรีด

**จุดเจ็บปวดในขั้นตอนนี้:**
ระหว่างการตัดจากเครื่องหล่อจนถึงการจัดกอง บิลเล็ตไม่มีตัวตนทางกายภาพ วิธีที่ใช้กันทั่วไปในปัจจุบันคือ "บันทึกตำแหน่ง + ลงหมายเลขเตาด้วยมือ" — เพียงลำดับการจัดกองถูกรบกวนหรือมีงานแทรก การระบุล็อตก็สูญหายทันที การพ่นรหัสไม่สามารถสร้างเครื่องหมายที่อ่านออกได้บนสเกล 750℃ ส่วนการแขวนป้ายด้วยมือบังคับให้พนักงานถือไม้แขวนป้ายเข้าใกล้บิลเล็ตร้อน

**HP-700T ทำงานอย่างไร:**
ติดตั้งสถานีหุ่นยนต์ติดฉลากหลังเครื่องตัด MES ส่งหมายเลขเตา → เครื่องพิมพ์ถ่ายโอนความร้อนพิมพ์ที่หน้างาน → หุ่นยนต์หยิบฉลาก → ติดอัตโนมัติบนหน้าตัดบิลเล็ต ชั้นกาวจะอ่อนตัวทันทีที่สัมผัสและซึมเข้าไปในรูพรุนหยาบของสเกล เมื่อเย็นตัวลงจะเกิดการยึดเชิงกล ตลอดช่วงที่บิลเล็ตเย็นตัวบนเตียง ถูกเครนเคลื่อนย้าย และวางในกอง บาร์โค้ดยังสแกนได้ต่อเนื่อง ก่อนเข้าเตา สถานีสแกนจะยืนยันตัวตนและส่งข้อมูลกลับไปยัง MES

## 7. CASE STUDY

**สายการหล่อต่อเนื่องของโรงงานเหล็กแห่งหนึ่ง · การติดโดยตรงบนบิลเล็ตร้อน**

- **ที่มา:** หลังการตัด บิลเล็ตถูกบริหารจัดการด้วยการบันทึกตำแหน่ง งานแทรกและการจัดกองที่สลับลำดับทำให้การระบุล็อตผิดพลาดบ่อยครั้ง
- **ความท้าทาย:** วัดได้ 750℃ ที่จุดติด และมีสเกลหลุดร่อนอย่างต่อเนื่อง
- **โซลูชัน:** สถานีหุ่นยนต์ติดฉลากหลังเครื่องตัด ใช้ HP-700T ร่วมกับการพิมพ์ถ่ายโอนความร้อนที่หน้างาน และสถานีสแกนก่อนเข้าเตา
- **ผลลัพธ์:** ยกเลิกการแขวนป้ายด้วยมือในเขตความร้อนสูง ความผิดพลาดในการระบุล็อตบิลเล็ตลดลงอย่างชัดเจน บันทึกการป้อนเตาส่งกลับ MES โดยอัตโนมัติ
