## 2. 钢铁行业为什么需要耐高温标签

钢铁生产的特殊性在于:**物料在被识别的那一刻,往往还是红热的、带氧化皮的、即将进入更严苛工序的。**

普通标签在这里全部失效:

- **纸 / PET 标签** — 200℃ 以上直接碳化
- **喷码 / 油漆** — 附着在氧化皮上,氧化皮一脱落标识随之消失;经加热炉后完全烧尽
- **人工挂牌** — 需要人员靠近 700℃ 热源,效率低且存在安全风险
- **钢印 / 打点** — 无法机读,无法与 MES 实时对接

耐高温标签的价值,是让**条码在整个热加工链条中始终保持可机读**,把物理物料和信息系统真正绑定。

## 3. 钢铁行业的标签应用类型

### 类型一:热态直贴标签(Direct Hot Application)

- **用在哪:** 连铸出坯后的板坯、钢坯,热轧后的钢卷
- **核心挑战:** 贴附瞬间对象表面 550–750℃,表面粗糙、覆盖氧化皮,标签必须瞬间贴牢且不燃烧
- **怎么用:** 由机器人贴标系统在辊道旁自动贴附,人员完全不接触热源
- **解决什么:** 热态物料从产出那一刻起就有唯一身份,不再靠"记位置""估炉次"

### 类型二:热处理过程标签(Through-Process Labels)

- **用在哪:** 线材 / 棒材的酸洗与退火、锻件的热锻与长时间退火
- **核心挑战:** 标签不是"耐一下高温",而是要**先泡酸(H₂SO₄ / HCl)、再进炉烧 10~18 小时**,化学腐蚀与长时间高温叠加
- **怎么用:** 常温下贴附,标签随工件走完整个热处理链,出炉后仍可扫码
- **解决什么:** 热处理是最容易丢批次的环节,标签活下来 = 追溯不断链

### 类型三:热处理挂牌(Tags)

- **用在哪:** 反复循环使用的工装、夹具、料筐、模具
- **核心挑战:** 不是一次性使用,而是要承受**几十上百次热循环**仍可读
- **怎么用:** 机械固定(绑扎 / 铆接)在资产上,长期跟随
- **解决什么:** 工装资产管理、使用次数统计、寿命预警

### 类型四:现场打印与系统集成

- **用在哪:** 贯穿以上所有场景
- **内容:** 热转印打印机 + 高温色带,由 MES 实时下发炉次 / 批号,打印后即时贴附;下游扫码站回传数据

## 2. 具体应用(Application)

**场景:** 连铸机切断后的钢坯,端面 / 侧面温度约 750℃,表面覆盖氧化皮并持续脱落。钢坯需在冷床等待、转运、堆垛,最终进入加热炉或直送轧机。

**这一步的痛点:**
连铸切断到堆垛之间,钢坯没有物理身份。目前普遍靠"记录位置 + 人工登记炉次",一旦堆垛顺序被打乱或有插单,批次归属就丢了。喷码在 750℃ 氧化皮上无法成型;人工挂牌需要操作工携带挂牌杆靠近热坯。

**HP-700T 的做法:**
在切断机后方设置机器人贴标工位。MES 下发炉次号 → 热转印打印机现场打印 → 机器人取标 → 在钢坯端面自动贴附。标签胶层在接触瞬间软化、渗入氧化皮的粗糙孔隙,冷却后形成机械锚固。钢坯在冷床降温、行车转运、堆垛期间,条码持续可扫;入炉前由扫码站确认身份并回传 MES。

**贴附位置建议:** 钢坯端面(氧化皮相对较薄、平整度较好),避开切割毛刺区。
